- Two fused operators with the pattern: `(data <op0> operand0) <op1> operand1`
- Broadcast operations over the free dimension via vector tiles

The fused computation becomes:
1. Compute `inv_std = 1/sqrt(var + epsilon)` using `nl.rsqrt`

4. Use `tensor_scalar(input_sb, op0=nl.multiply, operand0=scale, op1=nl.add, operand1=bias)` to fuse the final normalization

This reduces:
- **From ~5N+ Vector Engine cycles** (subtract + multiply + rsqrt + multiply + add, each with their own loop)
- **To ~2N+ Vector Engine cycles** (rsqrt + 2 fused vector operations for scale/bias computation + 1 fused tensor_scalar)
